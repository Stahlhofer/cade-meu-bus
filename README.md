# 🚌 Cadê Meu Bus?

Um aplicativo desenvolvido em **Flutter** que facilita a vida dos universitários fornecendo a localização aproximada em tempo real dos ônibus universitários nas rotas entre Panambi e Ijuí.

## 📱 Objetivo do Projeto

O **Cadê Meu Bus?** surge para resolver um dos principais desafios enfrentados pelos estudantes universitários: saber onde estão os ônibus de transporte. Com uma interface intuitiva e amigável, o aplicativo oferece:

- 📍 **Localização em Tempo Real**: Visualize a posição aproximada dos ônibus no mapa
- 🔔 **Atualizações Automáticas**: Receba informações constantemente atualizadas através do protocolo **MQTT**
- ⏱️ **Estimativa de Tempo**: Saiba aproximadamente quanto tempo levará para o ônibus
- 📊 **Histórico de Viagens**: Acompanhe o trajeto percorrido pelos ônibus

## 🛠️ Tecnologias Utilizadas

- **Flutter**: Framework multiplataforma para desenvolvimento de aplicativos iOS, Android, Web, Windows, macOS e Linux
- **MQTT**: Protocolo de mensageria eficiente para transmissão em tempo real da localização dos ônibus
- **Dart**: Linguagem de programação moderna usada pelo Flutter
- **GPS**: Integração com serviços de localização para rastreamento preciso
